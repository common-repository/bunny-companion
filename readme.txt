== Description ==

This is a plugin created for Bunny themes. This plugin provides additional frontpage sections for Bunny Themes.


== Installation ==

= Option 1. =

- Upload the entire `bunny-companion` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Bunny Companion" —> Click install.
- After plugin is downloaded, click active plugin



== Frequently Asked Questions ==
